BeautifulSoup

BeautifulSoup is a Python library for parsing HTML documents. We can use it to extract data from HTML we fetch from the web. Here, we're just using it to parse some simple sample HTML.

Searching the tree

BeautifulSoup allows us to search the HTML tree in lots of different ways: by tag, by id, by CSS class, and so on.

  1. By tag: we could search for every li
  2. By id: we could search for the p tag with id="searching_description"
  3. By class: we could search for every tag with a given class
  4. and so on...

Div tags are used to group block-elements and structure the web page.
